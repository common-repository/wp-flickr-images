Random flickr images on the basis of keywork in pages/ posts

== Description ==

This plugin help you add flickr images automatically to your wordpress page and post on the basis of provided keywork, you need to have flickr api key and secret which you get from your flicker account. it retrieve high quality image ( which is release under CC-SA licensed ) and randomize them to show different image each time.



== Installation ==

* Install the plugin by uploading via wordpress admin.
** OR **
* Upload all plugin files to your plugin directory via ftp
* And activate it from wordpress admin.
